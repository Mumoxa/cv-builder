# Mumoxa CV Builder

Mumoxa CV Builder is an Android-first, voice-guided CV creation app for South African frontline, entry-level, junior admin, junior finance, office support, and practical workers.

It is not just a CV template tool. It is a guided employability profile builder that helps a user create a better CV, understand what is missing, add evidence, and export a usable CV.

## Core direction

- Android-first mobile experience.
- Free CV creation for job seekers
