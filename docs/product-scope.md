# Product Scope

## Product name

Mumoxa CV Builder.

## What this product is

An Android-first, voice-guided CV and employability profile builder for South African frontline, entry-level, junior admin, junior finance, office support, and practical workers.

The product helps users build a usable CV even when they do not know how to describe their own work experience properly.

## What this product is not

This is not a basic CV template website.

This is not a recruiter
