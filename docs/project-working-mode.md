# Project Working Mode

This repository is the active working repository for the Mumoxa CV Builder app.

## Repository boundary

- Use `Mumoxa/cv-builder` for CV Builder work.
- Do not mix this project with Tallo, Careerize, or any other project unless the user clearly asks for that.
- Keep decisions and documentation aligned to the CV Builder strategy.
- Prefer small, clear commits that can be reviewed later.
- Do not claim that a feature is built unless it exists in the repository.
